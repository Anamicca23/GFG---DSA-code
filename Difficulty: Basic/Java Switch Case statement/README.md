<h2><a href="https://www.geeksforgeeks.org/problems/java-switch-case-statement3529/1">Java Switch Case statement</a></h2><h3>Difficulty Level : Difficulty: Basic</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18px;">Given an integer choice denoting the choice of the user and a list&nbsp;containing the single value R or two values&nbsp;L and B depending on the choice.<br>If the user's choice is 1, calculate the area of the circle having the given radius(<strong>R</strong>). &nbsp;<br>Else if the choice is 2, calculate the area of the rectangle with the given length(<strong>L</strong>) and breadth(<strong>B</strong>).</span></p>
<p><span style="font-size: 18px;"><strong>Note :</strong> A</span><span style="font-size: 18px;"> list arr[], containing the single value <strong>R </strong>or the two values <strong>L </strong>and <strong>B</strong>, as input parameters. Return the <strong>area</strong> of the desired geometrical figure. </span><span style="font-size: 18px;">Use Math.PI for the value of pi.</span></p>
<p><strong><span style="font-size: 18px;">Examples :</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> choice = 1, R = 5
<strong>Output:</strong> 78.53981633974483
<strong>Explaination:</strong> The choice is 1. So we have to calculate the area of the circle.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> choice = 2, L = 5, B = 10
