<h2><a href="https://www.geeksforgeeks.org/problems/total-number-of-spanning-trees-in-a-graph/1">Count Spanning Trees in a Graph</a></h2><h3>Difficulty Level : Difficulty: Hard</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a <strong>connected undirected graph</strong> with <strong>n</strong> vertices and<strong> m&nbsp;</strong>edges, where each edge is represented as edges[i]=[u,v]&nbsp; indicating an edge between vertices u and v. </span></p>
<p><span style="font-size: 18px;">Determine the total number of <strong>distinct spanning trees</strong> that can be formed from the graph.</span></p>
<p><span style="font-size: 18px;"><strong>Note:&nbsp;</strong></span><span style="font-size: 18px;">A spanning tree is a subgraph of the given graph that includes all n vertices, has exactly n-1 edges, is connected, and contains no cycles; therefore, every connected undirected graph always has at least one spanning tree.</span></p>
<p><span style="font-size: 14pt;"><span style="font-size: 12pt;"><strong style="font-size: 18px;">Examples:</strong></span></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 6, edges = [[0, 3], [0, 1], [1, 2], [1, 5], [3, 4]]<br><strong>Output:</strong> 1
<strong>Explanation: </strong></span><span style="font-size: 18px;">The graph has 6 vertices and 5 edges, and it is connected, so it is already a tree (m = n-1). A tree has only one spanning tree, which is the graph itself, so the answer is 1.<br>  <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/925219/Web/Other/blobid0_1777010882.png" width="221" height="205"><br></span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 3, edges = [[0, 1], [0, 2], [1, 2]]<br><strong>Output:</strong> 3
<strong>Explanation: </strong>There are exactly 3 possible spanning trees for the given graph. <br> <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/925219/Web/Other/blobid1_1777011785.png" width="614" height="168"><br></span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 1, edges = []<br><strong>Output:</strong> 1
<strong>Explanation: </strong></span><span style="font-size: 18.6667px;">With 1 vertex, a spanning tree needs 0 edges. The graph already satisfies this, so the answer is 1.</span></pre>
<p><strong style="font-size: 18px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">Constraints:<br></strong><span style="font-size: 14pt;">1 ≤ n </span><span style="font-size: 18.6667px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">≤</span><span style="font-size: 18.6667px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">&nbsp;10<sup><br></sup><span style="font-size: 14pt;">n -1&nbsp; ≤ m </span><span style="font-size: 18.6667px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">≤</span><span style="font-size: 18.6667px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"> n*(n-1)/2<br></span><span style="font-size: 14pt;">0 ≤ edges[i][0], edges[i][1] </span><span style="font-size: 18.6667px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">≤</span><span style="font-size: 18.6667px; font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;"> n-1</span></span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Graph</code>&nbsp;<code>Mathematical</code>&nbsp;<code>Tree</code>&nbsp;
