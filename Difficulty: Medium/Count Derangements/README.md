<h2><a href="https://www.geeksforgeeks.org/problems/dearrangement-of-balls0918/1">Count Derangements</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18.6667px;">Given a number&nbsp;<strong>n</strong>, find the total number of&nbsp;</span><span style="font-size: 18.6667px;">Derangements </span><span style="font-size: 18.6667px;">of elements from 1 to n. </span><span style="color: #273239; font-family: Nunito, sans-serif; font-size: 18px; letter-spacing: 0.162px; background-color: #ffffff;">A <strong>Derangement</strong> is a permutation of n elements, such that no element appears in its original position, i.e., 1 should not be the first element, 2 should not be second, etc. For example, [5, 3, 2, 1, 4] is a&nbsp;</span><span style="font-size: 18.6667px;">Derangement of first 5 elements.</span></p>
<p><span style="font-size: 14pt;"><strong>Note:</strong>&nbsp;The answer will always fit into a 32-bit integer.</span></p>
<p><strong><span style="font-size: 18px;">Examples:</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> n = 2
<strong>Output:</strong> 1
<span style="font-size: 14pt;"><strong>Explanation:</strong> F</span></span><span style="font-size: 14pt;">or [1, 2], there is only one possible derangement: [2, 1].</span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> n = 3
<strong>Output:</strong> 2
<strong>Explanation:</strong> For the set [1, 2, 3], there are only two possible derangements: [2, 3, 1] and [3, 1, 2].</span></pre>
