<h2><a href="https://www.geeksforgeeks.org/problems/mother-vertex/1">Mother Vertex</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a <strong>directed</strong> graph with <strong>V</strong> vertices labeled from 0 to V-1 and a list of edges <strong>edges[][]</strong>, where each edge is represented as [u, v] indicating a directed edge from vertex u to vertex v, find a <strong>Mother Vertex</strong> of the graph.</span></p>
<p><span style="font-size: 18px;">A Mother Vertex is a vertex from which all other vertices can be reached.</span></p>
<ul>
<li><span style="font-size: 18px;">If multiple such vertices exist, return the one with the <strong>smallest</strong> value.</span></li>
<li><span style="font-size: 18px;">If no such vertex exists, return <strong>-1</strong>.</span></li>
</ul>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>V = 5, edges[][] = [[0, 2], [0, 3], [1, 0], [2, 1], [3, 4]]</span><br><span style="font-size: 18px;"><strong>Output: </strong>0</span><br><span style="font-size: 18px;"><strong>Explanation: </strong>Vertices 0, 1, and 2 can each reach all other vertices in the graph. Among them, 0 is the smallest, so the output is 0.<br>  <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/701116/Web/Other/blobid0_1745298210.jpg" width="293" height="238"></span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>V = 3, edges[][] = [[0, 1], [2, 1]]</span>
<strong><span style="font-size: 18px;">Output: </span></strong><span style="font-size: 18px;">-1</span><br><span style="font-size: 18px;"><strong>Explanation: </strong>No vertex can reach all other vertices in the graph. Hence, there is no Mother Vertex, and the output is -1.<br></span> <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/701116/Web/Other/blobid1_1745298243.jpg" width="270" height="220"></pre>
<p><span style="font-size: 18px;"><strong>Constraints:</strong><br>1 ≤ V ≤ 10<sup>5</sup><br>1&nbsp;</span><span style="font-size: 18px;">≤ edges[i][0], edges[i][1] </span><span style="font-size: 18px;">≤ V-1</span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>DFS</code>&nbsp;<code>Graph</code>&nbsp;<code>Data Structures</code>&nbsp;<code>Algorithms</code>&nbsp;
