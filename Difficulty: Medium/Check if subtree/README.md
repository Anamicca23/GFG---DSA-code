<h2><a href="https://www.geeksforgeeks.org/problems/check-if-subtree/1">Check if subtree</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given two binary trees with roots <strong>root1</strong> (for tree <strong>T</strong>) and <strong>root2</strong> (for tree <strong>S</strong>), each containing at most <strong>N</strong> nodes.&nbsp;Determine whether tree S is a subtree of tree T.</span></p>
<p><span style="font-size: 18px;">Return <strong>true</strong> if S is a subtree of T, otherwise return <strong>false</strong>.</span></p>
<p><strong><span style="font-size: 18px;">Note:&nbsp;</span></strong><span style="font-size: 18px;">A tree S is considered a subtree of T if there exists a node in T such that the subtree rooted at that node is identical to S. Two trees are identical if they have the same structure and the same node values.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>root1 = [1, 2, 3, N, N, 4], root2 = [3, 4]<strong><br></strong>     <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926607/Web/Other/blobid0_1776750994.png" height="100">       <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926607/Web/Other/blobid1_1776751004.png" width="92" height="89"> <br><strong>Output: </strong>true 
<strong>Explanation:</strong> In the tree rooted at root1, the subtree starting at node 3 is identical to the tree rooted at root2 (same structure and node values). Hence, root2 is a subtree of root1, so the output is true.
</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>root1 = [26, 10, N, 20, 30, 40, 60], root2 = [26, 10, N, 20, 30, 40, 60]<br><strong>        <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926607/Web/Other/blobid0_1776751337.png" width="175" height="157">               <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926607/Web/Other/blobid1_1776751344.png" width="176" height="159"> </strong>
<strong>Output: </strong>true 
<strong>Explanation: </strong>Both root1 and root2 represent identical trees. So, root2 is a subtree of root1, and the output is true.
</span></pre>
