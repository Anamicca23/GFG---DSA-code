<h2><a href="https://www.geeksforgeeks.org/problems/longest-repeating-character-replacement/1">Longest Repeating Character Replacement</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a string <strong>s&nbsp;</strong>of length <strong>n</strong>&nbsp;consisting of uppercase English letters and an integer <strong>k</strong>, you are allowed to perform<strong> at most</strong> k operations.&nbsp; </span><span style="font-size: 18px;">In each operation, y</span><span style="font-size: 18px;">ou can change any character of the string to any other uppercase English letter.</span></p>
<p><span style="font-size: 18px;"> Determine the length of the <strong>longest substring</strong> that can be transformed into a string with all identical characters after performing at most k such operations.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><span style="font-size: 18px;"><strong>Input</strong>: s = "ABBA", k = 2 <br></span></span><span style="font-size: 18px;"><span style="font-size: 18px;"><strong>Output:</strong> 4 <br><strong>Explanation: </strong>The string "ABBA" can be fully converted into the same character using at most 2 changes. By replacing both 'A' with 'B', it becomes "BBBB". Hence, the maximum length is 4.<br></span></span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>s = "ADBD", k = 1
<strong>Output: </strong>3
<strong>Explanation: </strong>In the string "ADBD", we can make at most 1 change. By changing 'B' to 'D', the string becomes "ADDD", which contains a substring "DDD" of length 3.</span></pre>
<p><span style="font-size: 18px;"><strong>Constraints:<br></strong></span><span style="font-size: 18px;">1&nbsp;<span style="font-size: 18.6667px;">≤</span> n, k <span style="font-size: 18.6667px;">≤ </span>10<sup>5</sup><br>s consists of only uppercase English letters.</span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>two-pointer-algorithm</code>&nbsp;<code>Strings</code>&nbsp;<code>Data Structures</code>&nbsp;<code>Algorithms</code>&nbsp;
