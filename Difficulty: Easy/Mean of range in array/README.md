<h2><a href="https://www.geeksforgeeks.org/problems/mean-of-range-in-array2123/1">Mean of range in array</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18px;">Given an integer array <strong>arr[]</strong> and a 2D array <strong>queries[][]</strong>. Each query <strong>queries[i] = [l, r] </strong>represents a subarray ranging from index l to r (inclusive). </span><span style="font-size: 18px;">For every query, compute the <strong>mean (average) </strong>of the elements in the specified range, and return the <strong>floor value</strong> of that mean.</span></p>
<p><span style="font-size: 18px;">Return an array where each element corresponds to the result of a query.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> arr[] = [1, 2, 3, 4, 5], <span style="font-size: 14pt;"><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif; white-space-collapse: collapse;">queries</span>[][] = [[0, 2], [1, 3], [0, 4]]</span><br><strong>Output:</strong> [2, 3, 3]<br><strong>Explanation:</strong> </span><span style="font-size: 18px;">The array is [1, 2, 3, 4, 5].<br></span><span style="font-size: 18px;">Query 1: l = 0, r = 2 → subarray [1, 2, 3] → sum = 6 → mean = 6/3 = 2<br></span><span style="font-size: 18px;">Query 2: l = 1, r = 3 → subarray [2, 3, 4] → sum = 9 → mean = 9/3 = 3<br></span><span style="font-size: 18px;">Query 3: l = 0, r = 4 → subarray [1, 2, 3, 4, 5] → sum = 15 → mean = 15/5 = 3<br>Hence the answer is [2, 3, 3]</span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> arr[] = [6, 7, 8, 10], queries<span style="font-size: 14pt;">[][]</span> = [[0, 3], [1, 2]]<br><strong>Output:</strong> [7, 7]<br><strong>Explanation:</strong> The array is [6, 7, 8, 10].<br>Query 1: l = 0, r = 3 </span><span style="font-size: 14pt;"><span style="font-size: 14pt;">→ subarray [6, 7, 8, 10] → sum = 31 → mean = 31/4 = 7 (floor value)<br></span><span style="font-size: 18.6667px;">Query 1: l = 1, r = 2 → subarray [7, 8] → sum = 15 → mean = 15/2 = 7 (floor value)<br>Hence the answer is [7, 7]</span></span></pre>
<p><span style="font-size: 18px;"><strong>Constraints:</strong>&nbsp;<br>1 ≤ arr.size() ≤ 10<sup>5</sup><br>1 ≤ arr[i] ≤ 10<sup>3<br></sup>1 ≤&nbsp;queries.size() ≤ 10<sup>5</sup><br>1 ≤ queries[i][0]&nbsp;</span><span style="font-size: 18px;">≤</span><span style="font-size: 18px;">&nbsp;</span><span style="font-size: 18px;">queries[i][1]</span><span style="font-size: 18px;"> &lt; arr.size()</span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>prefix-sum</code>&nbsp;<code>Arrays</code>&nbsp;<code>Data Structures</code>&nbsp;<code>Algorithms</code>&nbsp;
