<h2><a href="https://www.geeksforgeeks.org/problems/palindrome-numbers0942/1">Palindrome Binary</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18.6667px;">Given an integer <strong>n</strong>, determine whether its binary representation forms a <strong>palindrome</strong>. </span><span style="font-size: 18.6667px;">Return <strong>true</strong> if the binary representation of n is a palindrome; otherwise, return <strong>false</strong>.</span></p>
<p><strong><span style="font-size: 18.6667px;">Note:&nbsp;</span></strong><span style="font-size: 18.6667px;">A binary representation is considered a palindrome if it reads the same forward and backward.</span></p>
<p><strong style="font-size: 18px;">Examples:</strong></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 17<br><strong>Output:</strong> true
<strong>Explanation:</strong> Binary representation of 17 is (10001)<sub>2, </sub>which reads the same forward and backward, so it is a palindrome. </span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> n = 16<br><strong>Output:</strong> false
<strong>Explanation: </strong>Binary representation of 16 is (10000)<sub style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">2, </sub><span style="font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Oxygen, Ubuntu, Cantarell, 'Open Sans', 'Helvetica Neue', sans-serif;">which is not a palindrome. </span><br></span></pre>
