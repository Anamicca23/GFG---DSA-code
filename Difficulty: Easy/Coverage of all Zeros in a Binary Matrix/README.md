<h2><a href="https://www.geeksforgeeks.org/problems/coverage-of-all-zeros-in-a-binary-matrix4024/1">Coverage of all Zeros in a Binary Matrix</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="color: #273239; font-family: Nunito, sans-serif;"><span style="font-size: 18px; letter-spacing: 0.162px;">Given a binary matrix <strong>mat[][]</strong> containing only <strong>0</strong>s and <strong>1</strong>s, find the total coverage of all <strong>0's</strong>.&nbsp;</span></span><span style="color: #273239; font-family: Nunito, sans-serif;"><span style="font-size: 18px; letter-spacing: 0.162px;">The coverage of a particular 0 cell is defined by checking 1's in its four directions (left, right, up, and down). For each direction, if there is at least one 1 anywhere between the 0 and the boundary of the matrix, the coverage increases by one.</span></span></p>
<p><span style="color: #273239; font-family: Nunito, sans-serif;"><span style="font-size: 18px; letter-spacing: 0.162px;">Return the sum of the coverage values for all 0 cells in the matrix.</span></span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input : </strong>mat[][] =<strong> </strong>[[1, 1, 1, 0],<br>                  [1, 0, 0, 1]]<strong><br>Output : </strong>8<strong><br>Explanation: </strong>Coverage of first zero is 2. Coverages of other two zeros is 3 Total coverage = 2 + 3 + 3 = 8<br></span><img style="max-width: 98%; margin: 1%;" src="https://media.geeksforgeeks.org/wp-content/uploads/20260515141311357568/2056958046.webp" alt="2056958046"></pre>
<pre><strong><span style="font-size: 14pt;">Input: </span></strong><span style="font-size: 14pt;">mat[][] = [[0, 1, 0],<br>               [0, 1, 1],</span><br><span style="font-size: 14pt;">               [0, 0, 0]]<br></span><span style="font-size: 14pt; font-family: Arial, sans-serif; font-variant-numeric: normal; font-variant-east-asian: normal; font-variant-alternates: normal; font-variant-position: normal; vertical-align: baseline; white-space-collapse: preserve;"><strong>Output</strong>: 6<br></span><span style="font-size: 14pt; font-family: Arial, sans-serif; font-variant-numeric: normal; font-variant-east-asian: normal; font-variant-alternates: normal; font-variant-position: normal; vertical-align: baseline; white-space-collapse: preserve;"><strong>Explanation: </strong>Total Coverage is 1 + 2 + 1 + 0 + 1 + 1 = 6   </span></pre>
